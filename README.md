# F1 2026 Australia GP Analysis

My first GitHub project! 🏎️

This script analyzes the 2026 Formula 1 Australian Grand Prix using FastF1.

## What it does
- Race lap time comparison (all drivers)
- Tyre strategy visualization
- Qualifying sector time delta comparison

## Requirements
pip install fastf1 matplotlib pandas numpy

## Usage
python f1_2026_australia_analysis.py

## Output
- f1_race_laptimes.png
- f1_tyre_strategy.png
- f1_quali_sectors.png

## Data Source
Data retrieved via FastF1 library from the official F1 API.
